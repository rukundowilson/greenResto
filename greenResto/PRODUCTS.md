## How to View Products

1. **Build the project:**
   ```bash
   npm run build
   ```

2. **Start the server:**
   ```bash
   npm run serve
   ```

3. **Open in browser:**
   - Navigate to `http://localhost:8080`
   - All 9 products will be displayed in a grid layout
   - Each product shows:
     - Product image
     - Category name (bold)
     - Full product name
     - Price
     - "Add to Cart" button
