
## Setup
1. Install dependencies:
```bash
npm install
```

2. Build TypeScript:
```bash
npm run build
```

3. Serve the application:
```bash
npm run serve
```
npm run dev

Or use any static file server to serve the root directory.

## Development

Watch mode for TypeScript:
```bash
npm run dev
```

## Features

- TypeScript with strict mode
- Vanilla JavaScript (no frameworks)
- Single page application
- Responsive design
- Immutable cart operations
- Type-safe code
