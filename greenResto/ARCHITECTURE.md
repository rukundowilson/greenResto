# Architecture Flow

## How Products Are Rendered

```
┌─────────────────────────────────────────────────────────┐
│ 1. TypeScript Source Code (src/)                        │
│    - products.ts: Product data objects                  │
│    - main.ts: Rendering logic                           │
└─────────────────┬───────────────────────────────────────┘
                  │
                  │ npm run build (TypeScript Compiler)
                  ▼
┌─────────────────────────────────────────────────────────┐
│ 2. Compiled JavaScript (dist/)                         │
│    - products.js: Product data                          │
│    - main.js: Rendering functions                       │
└─────────────────┬───────────────────────────────────────┘
                  │
                  │ Browser loads HTML
                  ▼
┌─────────────────────────────────────────────────────────┐
│ 3. HTML File (index.html)                               │
│    - Empty containers: <div id="desserts-section">     │
│    - Script tag: <script src="dist/main.js">           │
└─────────────────┬───────────────────────────────────────┘
                  │
                  │ Page loads, JavaScript executes
                  ▼
┌─────────────────────────────────────────────────────────┐
│ 4. Runtime (Browser)                                    │
│    - JavaScript reads products array                    │
│    - Generates HTML for each product                    │
│    - Inserts HTML into containers                       │
│    - Result: 9 product cards displayed                  │
└─────────────────────────────────────────────────────────┘
```

## Step-by-Step Process

1. **Write TypeScript** (`src/data/products.ts`)
   ```typescript
   export const products: Dessert[] = [
     { id: "waffle", name: "Waffle with Berries", ... },
     // ... 8 more products
   ];
   ```

2. **Compile to JavaScript** (`npm run build`)
   - TypeScript → JavaScript
   - Output: `dist/data/products.js`

3. **HTML loads JavaScript** (`index.html`)
   ```html
   <script type="module" src="dist/main.js"></script>
   ```

4. **JavaScript runs on page load**
   ```javascript
   // main.js runs automatically
   function init() {
     renderDesserts(); // Reads products array
     renderCart();
   }
   ```

5. **Products rendered dynamically**
   ```javascript
   // JavaScript generates HTML and inserts it
   dessertsSection.innerHTML = products.map(product => `
     <div class="dessert-card">
       <img src="${product.image.thumbnail}">
       <h3>${product.name}</h3>
       ...
     </div>
   `).join('');
   ```

## Key Points

- ✅ **No TypeScript in HTML** - It's compiled to JavaScript first
- ✅ **Dynamic Rendering** - Products are generated by JavaScript
- ✅ **Single Page Application** - All content loaded via JavaScript
- ✅ **Type Safety** - TypeScript ensures correct data structure
